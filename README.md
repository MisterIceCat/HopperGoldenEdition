# HopperGoldenEdition

This mod is a backport of the Golden Hopper mod for Minecraft 1.12.2.
The original mod was made by MrCrayfish.

This mod backport Golden Hopper to Minecraft 1.12.2
and keep it as close as possible to the original mod.

The original mod: https://www.curseforge.com/minecraft/mc-mods/golden-hopper

Creators:
MrCrayfish for the original Golden Hopper mod.
This project is not official and not made by MrCrayfish.
It's backported by MisterIceCat and ACGaming.
